# Chinese Name Generator

A web application that generates meaningful Chinese names for foreigners.

## Features

- Generate Chinese names based on English names
- Provides pinyin (pronunciation) for each name
- Explains the meaning and cultural significance of the generated names

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- API: Zhipu AI GLM-4
- Deployment: Vercel
